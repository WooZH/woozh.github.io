写一个简单的Vue组件

发表于 2017-11-30

组件是什么

以下是vue作者对组件的描述

组件 (Component) 是 Vue.js 最强大的功能之一。组件可以扩展 HTML 元素，封装可重用的代码。在较高层面上，组件是自定义元素，Vue.js 的编译器为它添加特殊功能。

在组件的定义中，我们可以看到封装可重用的代码这个点，这个点也是我们最常用的出发点。

在许多完善的组件库中，我们可以看到许多样式美观、可配置、用法方便的组件，例如：

  • 按钮
  • 表单
  • 导航栏
  • 对话框
  • …

这些组件可以帮助我们快速搭建系统，风格美观又统一，减少了大量的HTML+CSS的工作。

这样的组件就是我们需要的组件。

如果需要详细了解，请点击链接，阅读文档。

什么时候用组件

例如在我们的后台管理系统中，所有的页面都有一个侧边导航栏。按照以前的开发方式，我们需要给十几个内容页，写十几个几乎相同的侧边导航栏。

这样的导航栏，我们就应该把它做成一个公用的组件。这个组件包含了基本的html结构和样式，父组件只需要传入一个list的数据，组件就能够渲染出一个美观又统一的侧边栏。

单向数据流

Prop 是单向绑定的：当父组件的属性变化时，将传导给子组件，但是反过来不会。

父子组件通信（自定义事件）

既然数据流是单向的，那么当子组件需要改变父组件的值该怎么做呢？

  • 使用 $on(eventName) 监听事件
  • 使用 $emit(eventName) 触发事件

可复用的组件

Vue组件由API来自三个部分：

  • Prop 允许外部环境传递数据给组件；
  • 事件允许从组件内触发外部环境的副作用；
  • 插槽允许外部环境将额外的内容组合在组件中。
